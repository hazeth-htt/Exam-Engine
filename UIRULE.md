# UI Rules

NEVER

- Use gradient
- Use glassmorphism
- Use neumorphism
- Use dashboard templates
- Use Bootstrap style
- Use Material UI style
- Put icons in every button
- Use colorful badges everywhere
- Use excessive borders
- Use nested cards
- Use tables for browsing folders
- Use floating action buttons
- Use hero banners
- Use decorative illustrations

ALWAYS

- Use whitespace
- Use typography hierarchy
- Use Finder-like navigation
- Use breadcrumb
- Use contextual actions
- Use context menu
- Use subtle animations
- Use neutral palette
- Use consistent spacing
- Use reusable components
